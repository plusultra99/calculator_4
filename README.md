# calculator_4
PyQt5 based calculator - used for practice

## Goals

* 오픈소스 소프트웨어 github 저장소 생성 및 관리 실습
* PyQt5 기반 계산기 기능 및 사용방법 개선

## Getting Started

* PyQt5 모듈 설치 (python -m pip install pyqt5)
* 파이썬 파일 실행 (python calculator_main.py)
